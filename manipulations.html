Manipulations

These days software exists from Premium packages like Photoshop to freeware software like The GIMP which allow users a crazy amount of power over their images, too much to cover here but most people will be eternally content with basic edits such as:

  1. Make yourself look younger

    The healing brush in PhotoShop uses a similar method to interpolation schemes to fix wrinkles and the such like.

  2. Color Correction

    Auto Contrast and auto Color are easy ways to 'correct' contrast (black/white) levels and color to levels which go better together creating a more vibrant image.

  3. Removing Red Eye

    Sometimes we all look like the devil in certain pictures, I have in the past removed the red eye effect using Googles' freeware tool picasa

  4. Whitening Teeth

    The Sponge tool in PhotoShop changes saturation of colors making it perfect for such a task.

  5. Brighten Up a Specific Part of an Image

    Using the Levels tool

  6. Create Easy Reflections of Any Image

    Image > Transform

  7. Extract an object and put Them on a New Background

    The Magic Wand tool can do this if your foreground and background are different enough.

    There as supposedly other ways more suitable ways to do this e.g. Polygonal Lasso tool but I don't understand that.

  8. Fix Distortion of Images

    Liquify filter allow user to push, pull, rotate, reflect pixels

  9. Change a Specific Color

    Photoshop has a one-click color replacement tool.

  10. Apply arty hipster filters to images

    This has become all the range after apps like Pixl O Matic and Instagram soared in popularity
