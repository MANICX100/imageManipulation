Accessibility:

  1. All embedded images contain alternative text throughout these pages; to help users which may require a screen reader.

  2. Also All video content is universal for users of devices which may or may not support flash video.

  3. Has been tested to be browsable with only a keyboard.

  4. Is fully compatible with screen readers and text to speech engines.

  5. Unfortunately there is no workaround for lack of image support in certain browsers or CSS3 transformations hence why I have installed a warning system if you are using an out-of-date system.

  6. All current versions of major browsers are supported
